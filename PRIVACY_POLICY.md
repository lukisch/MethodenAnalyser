# Privacy Policy — MethodenAnalyser

**Last updated:** March 13, 2026
**Publisher:** Geiger

## Overview

MethodenAnalyser is a static Python code analysis tool with a graphical user interface. This privacy policy explains how the application handles your data.

## Data Collection

**MethodenAnalyser does not collect, store, or transmit any personal data to the developer.**

All data processing happens locally on your device. The application does not include analytics, telemetry, crash reporting, or any form of data collection.

## Data Processing

The application analyzes Python source files that you choose to open or scan. All code analysis — including detection of unused imports, duplicate code, and import scope issues — is performed entirely on your local machine. No source code, file paths, or analysis results are sent to the developer or any third party.

## External Services

MethodenAnalyser does not connect to any external services or the internet. It is a fully offline application.

## Local Storage

The application may store user preferences (e.g., window size, last opened directory) locally on your device. No data is synced to external servers.

## Children's Privacy

MethodenAnalyser does not knowingly collect any data from children or any other users.

## Changes to This Policy

Updates to this privacy policy will be posted on this page with an updated revision date.

# Datenschutzrichtlinie — MethodenAnalyser

**Zuletzt aktualisiert:** 13. März 2026
**Herausgeber:** Geiger

## Überblick

MethodenAnalyser ist ein statisches Python-Code-Analyse-Tool mit grafischer Benutzeroberfläche. Diese Datenschutzrichtlinie erläutert, wie die Anwendung mit Ihren Daten umgeht.

## Datenerhebung

**MethodenAnalyser erhebt, speichert oder überträgt keine personenbezogenen Daten an den Entwickler.**

Die gesamte Datenverarbeitung erfolgt lokal auf Ihrem Gerät. Die Anwendung enthält keine Analyse-, Telemetrie-, Absturzberichts- oder sonstige Datenerhebungsfunktionen.

## Datenverarbeitung

Die Anwendung analysiert Python-Quelldateien, die Sie öffnen oder scannen möchten. Alle Codeanalysen — einschließlich der Erkennung ungenutzter Imports, doppelter Code-Abschnitte und Import-Scope-Probleme — werden ausschließlich auf Ihrem lokalen Rechner durchgeführt. Kein Quellcode, keine Dateipfade oder Analyseergebnisse werden an den Entwickler oder Dritte übermittelt.

## Externe Dienste

MethodenAnalyser stellt keine Verbindung zu externen Diensten oder dem Internet her. Es handelt sich um eine vollständig offline nutzbare Anwendung.

## Lokale Speicherung

Die Anwendung kann Benutzereinstellungen (z. B. Fenstergröße, zuletzt geöffnetes Verzeichnis) lokal auf Ihrem Gerät speichern. Es werden keine Daten mit externen Servern synchronisiert.

## Datenschutz für Kinder

MethodenAnalyser erhebt wissentlich keine Daten von Kindern oder anderen Nutzern.
